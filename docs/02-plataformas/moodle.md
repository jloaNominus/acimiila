# Moodle

## Rol dentro de la arquitectura

Moodle es la plataforma de aprendizaje. Su función es sostener el LMS, la experiencia pedagógica, la gestión de contenidos y la interacción académica.

## Alcance

- Cursos y estructuras de aprendizaje.
- Steps, grupos, clases, tareas y evaluaciones.
- Actividades colaborativas y seguimiento didáctico.
- Datos y evidencias propias del proceso formativo.

## Lo que no debe cargar

- Lógica administrativa compleja.
- Modelos empresariales relacionales completos.
- Reglas contables o de ERP.
- Persistencia de procesos que ya deban vivir en AciMiila.

## Integración esperada

Moodle debe publicar o permitir consultar eventos útiles para AciMiila, pero sin perder su naturaleza de LMS. Su API futura será el punto de salida para el orquestador Moodle -> AciMiila.
